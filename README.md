# Context Engineering Resources

[![Master Index](https://img.shields.io/badge/Master%20Index-All%20Repositories-purple?style=for-the-badge&logo=github)](https://github.com/danielrosehill/Index)

A curated collection of resources on **context engineering** — the practice of designing, structuring, retrieving, and managing the context that large language models and AI agents operate over. Covers tooling (MCP servers, context stores, retrieval layers), agent skills, educational material, and awesome lists.

**Last Updated:** April 2026

---

## Highlights

A handful of repositories stand out by community traction in this space:

- **[Context7](https://github.com/upstash/context7)** — up-to-date code documentation injected into LLM context for AI coding tools.
- **[OpenViking](https://github.com/volcengine/OpenViking)** — open-source context database unifying agent memory, resources, and skills via a filesystem paradigm.
- **[Agent Skills for Context Engineering](https://github.com/muratcankoylan/Agent-Skills-for-Context-Engineering)** — large library of agent skills focused on context management for production agents.
- **[Figma Context MCP](https://github.com/GLips/Figma-Context-MCP)** — feeds Figma layout data to coding agents like Cursor.
- **[WeKnora](https://github.com/Tencent/WeKnora)** — Tencent's RAG framework for deep document understanding and context-aware retrieval.
- **[Context Engineering Intro](https://github.com/coleam00/context-engineering-intro)** — popular hands-on introduction centred on Claude Code.
