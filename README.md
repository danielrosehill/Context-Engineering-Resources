# Context Engineering Resources

[![Master Index](https://img.shields.io/badge/Master%20Index-All%20Repositories-purple?style=for-the-badge&logo=github)](https://github.com/danielrosehill/Index)

A curated collection of resources on **context engineering** — the practice of designing, structuring, retrieving, and managing the context that large language models and AI agents operate over. Covers tooling (MCP servers, context stores, retrieval layers), agent skills, educational material, and awesome lists.

**Last Updated:** April 2026

---

## Highlights

A handful of repositories stand out by community traction in this space:

- **[Context7](https://github.com/upstash/context7)** — up-to-date code documentation injected into LLM context for AI coding tools.
- **[OpenViking](https://github.com/volcengine/OpenViking)** — open-source context database unifying agent memory, resources, and skills via a filesystem paradigm.
- **[Agent Skills for Context Engineering](https://github.com/muratcankoylan/Agent-Skills-for-Context-Engineering)** — large library of agent skills focused on context management for production agents.
- **[Figma Context MCP](https://github.com/GLips/Figma-Context-MCP)** — feeds Figma layout data to coding agents like Cursor.
- **[WeKnora](https://github.com/Tencent/WeKnora)** — Tencent's RAG framework for deep document understanding and context-aware retrieval.
- **[Context Engineering Intro](https://github.com/coleam00/context-engineering-intro)** — popular hands-on introduction centred on Claude Code.

## Tools

MCP servers, context stores, retrieval layers, memory systems, libraries, and platforms.

### .context

A Git-native, AI-optimized documentation system that turns a repo into a living knowledge base for agents.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/andrefigueira/.context)

### ACAT (Assistive Context-Aware Toolkit)

Intel's Assistive Context-Aware Toolkit — context-driven assistive communication software (originally built for users with motor neurone disease). Included as an adjacent example of context-aware system design rather than LLM context engineering proper.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/intel/acat)

### Agentic Context Engine

Lets agents learn from experience and persist that learning across runs. Also offered as a hosted product at kayba.ai.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/kayba-ai/agentic-context-engine)

### Airweave

Open-source context retrieval layer designed to sit underneath AI agents.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/airweave-ai/airweave)

### Better Context

Pulls up-to-date context on the libraries and frameworks used in your project so coding agents work from current docs rather than stale training data.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/davis7dotsh/better-context)

### Claude Context

Code-search MCP for Claude Code that makes the entire codebase available as context to any coding agent.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/zilliztech/claude-context)

### CodeGraphContext

MCP server and CLI that index a local codebase into a graph database, exposing structural code context to AI assistants.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/CodeGraphContext/CodeGraphContext)

### Context

Local-first documentation system for AI agents.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/neuledge/context)

### Context-Engine

MCP-based agentic context compression suite.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/Context-Engine-AI/Context-Engine)

### Context Gateway

An agentic proxy that adds instant history compaction and context-optimization tools to any AI agent workflow.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/Compresr-ai/Context-Gateway)

### Context Hub

Andrew Ng's context-hub — a context platform for AI applications.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/andrewyng/context-hub)

### Context Infrastructure

A context and memory system for AI coding agents — persistent memory, personal rules, skills, and scheduled observations.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/grapeot/context-infrastructure)

### Context Mode

Context-window optimisation for AI coding agents — sandboxes tool output for a reported ~98% reduction; works across 12 platforms.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/mksglu/context-mode)

### Context Portal (ConPort)

A memory-bank MCP server that builds a project-specific knowledge graph for IDE-based RAG and context-aware development.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/GreatScottyMac/context-portal)

### Context7

Up-to-date code documentation for LLMs and AI code editors, available as an MCP server and hosted platform.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/upstash/context7)

### ContextGem

Library for effortless LLM-driven structured extraction from documents.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/shcherbak-ai/contextgem)

### ContextKeep

MCP server providing long-term memory storage for AI agents.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/mordang7/ContextKeep)

### Continuous Claude v3

Context-management system for Claude Code — uses hooks, ledgers, and handoffs to maintain state, plus MCP execution and orchestration with isolated context windows per agent.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/parcadei/Continuous-Claude-v3)

### dotai

A context manager intended to work across multiple agent frameworks.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/udecode/dotai)

### Figma Context MCP

MCP server that exposes Figma layout information to AI coding agents (Cursor, Claude Code, etc.) so they can act on design files.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/GLips/Figma-Context-MCP)

### FreeTodo

Auto-manages personal task context with AI — a task store designed to be agent-readable.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/FreeU-group/FreeTodo)

### Headroom

A context-optimization layer for LLM applications — sits between the app and the model to manage what gets sent.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/chopratejas/headroom)

### Long Context Attention (USP)

Unified (a.k.a. hybrid 2D) sequence-parallel attention implementation for training and inference of long-context transformer models.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/feifeibear/long-context-attention)

### Lossless Claw

LCM (Lossless Context Management) plugin for OpenClaw — preserves agent context across sessions without truncation.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/Martian-Engineering/lossless-claw)

### MCP Server for Qdrant

Official Qdrant Model Context Protocol server — exposes a Qdrant vector store as a context source for MCP-aware agents.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/qdrant/mcp-server-qdrant)

### OpenContext

Personal context store for AI agents and assistants — capture, search, and reuse project knowledge across Codex, Claude, OpenCode, etc., with a desktop GUI.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/0xranx/OpenContext)

### OpenViking

Open-source context database designed for AI agents (e.g. OpenClaw). Unifies management of memory, resources, and skills via a filesystem paradigm with hierarchical context delivery.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/volcengine/OpenViking)

### TrustGraph

Context-development platform — store, enrich, and retrieve structured knowledge with graph-native infrastructure, semantic retrieval, and portable context cores.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/trustgraph-ai/trustgraph)

### WeKnora

Tencent's LLM-powered RAG framework for deep document understanding, semantic retrieval, and context-aware answering.

[![View Repo](https://img.shields.io/badge/View-Repo-blue?style=flat&logo=github)](https://github.com/Tencent/WeKnora)
